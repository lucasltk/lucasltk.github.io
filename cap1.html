OPERADORES RELACIONAIS

São utilizados para comparar caracteres e números. Os valores a serem comparados podem ser caracteres ou variáveis. Sempre retornam valores lógicos (verdadeiro ou falso);

operadores: Igual a: == Diferente de: < > Maior que: > Menor que: < Maior ou igual a: >= Menor ou igual a: <=
Exemplo:
Variável A= 5;
Variável B=8;
Os resultados são:
A = B : Falso
A<>B : Verdadeiro
A > B: Falso
A < B: Verdadeiro
A >= B : Falso.
A <= B: Verdadeiro.

OPERADORES ARITMÉTICOS

São utilizados para obter resultados numéricos (adição, subtração, multiplicação, divisão e exponenciação).

Símbolos aritméticos:

Adição -> + Subtração-> -
Multiplicação-> * Divisão: / Exponênciação -> **

VARIÁVEIS

Variáveis e constantes são os elementos básicos que um programa manipula. Uma variável é uma espaço na memória do computador para armazenar um tipo de dado determinado. Variáveis devem receber nomes para poderem ser referenciadas e modificadas quando necessário. Um programa deve conter declarações que especificam de que tipo são as variáveis que serão utilizadas e as vezes com valor inicial.

Tipos de dados:
  • Inteiros -> 1; 2; 3; 0; -1; -5; 12; 100; .....
  • Reais -> 1,25; 3,24; 12;8; 0,26; ....
  • Caracteres -> "João", "Treinar algoritmos", "V" ;
  • Lógica -> Verdadeiro ou Falso;

CONSTRUÇÃO DO ALGORITMO

Deve-se descrever uma sequência lógica de instruções:

  • Usar somente um verbo por frase
  • Imaginar que você está desenvolvendo para pessoas que não trabalham com informática
  • Usar frases curtas e simples
  • Ser objetivo

O algoritmo é dividido em fases:

  • Entrada: São os dados de entrada do algoritmo.
  • Processamento: São os procedimentos utilizados para chegar ao resultado final.
  • Saída: Dados já processados.

PSEUDOCÓDIGO

Os algoritmos são escritos em uma linguagem de programação chamada pseudocódigo. Isso é uma alusão à uma posterior linguagem de programação (Java, Visual Basic, C#, PHP), ou seja, quando for programar em java, usaremos código java, se for em PHP, usaremos código PHP. Logo, a lógica de programação independe de sua linguagem.

O código deve ser fácil de se interpretar e fácil de codificar, deve ser o intermediário entre a linguagem falada e a linguagem de programação.

Introdução

A lógica de programação é necessária para pessoas que desejam trabalhar com desenvolvimento de sistemas e programas, ela permite definir a seqüência lógica para o desenvolvimento. Então o que é lógica? Lógica de programação é a técnica de encadear pensamentos para atingir determinado objetivo.

A forma de colocar esses pensamentos em prática se dá atráves dos algoritmos. Um algoritmo é uma sequência de passos para se executar uma função. Um exemplo de algoritmo, fora da computação, é uma "receita de bolo".

Exemplo: "Mascar Chicletes"
  • 1. Pegar chicletes.
  • 2. Retirar o papel.
  • 3. Mascar o chicletes.
  • 4. Jogar papel no lixo.
